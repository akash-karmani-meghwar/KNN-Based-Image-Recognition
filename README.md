**KNN Image Classification**

This project loads image data from a MATLAB file using Python.

**Install Libraries**

Run this in your terminal:

```bash
pip install numpy scipy
```

**Files**

- `data.mat` (Dataset)
- `main.py` (Code)

**How to Run**

1. Open the folder in VS Code.
2. Check the file path inside `main.py` and update it to your folder location.
3. Run this command:

bash
python main.py
